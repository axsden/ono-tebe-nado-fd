# Описание

Одностраничный сайт аукциона "Оно тебе надо". Проектная работа выполнена в рамках обучения в Яндекс Практикуме.


## Технологии

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Семантическая верстка
- Методология именования классов по БЭМ

## Структура проекта

```
ono-tebe-nado/
├── fonts/             # Шрифты Raleway (Regular, Bold)
│   ├── fonts.css      # Подключение шрифтов
│   ├── Raleway-Regular.woff2
│   ├── Raleway-Regular.woff
│   ├── Raleway-Bold.woff2
│   └── Raleway-Bold.woff
├── images/            # Изображения проекта
├── styles/
│   ├── global.css     # Сброс браузерных стилей
│   └── style.css      # Основные стили
├── index.html         # Главная страница
```
